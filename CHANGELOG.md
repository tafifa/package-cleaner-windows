## [0.1.0] - 2026-05-21

### Added

- Baseline PackClean desktop app (Electron + React + Vite).
- Project scanner for JS/PHP/Flutter projects.
- Recursive dependency folder size calculation.
- Multi-select and bulk actions.
- Recycle Bin based delete flow.
- Reinstall commands per project type.
- Root folder and size-threshold settings.
- Terminal output panel.

### Changed

- Improved UI hierarchy and grouped filter layout.
- Floating bulk action bar for better scroll-time access.
- Added `Clear Selected` action.
- Added in-app delete confirmation modal with path preview.
